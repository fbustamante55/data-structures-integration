# Data Structures Integration

A Java project that integrates and demonstrates multiple data structures including linked lists, stacks, queues, and trees within a single application.

## Technologies
- Java 11+
- Data Structures & Algorithms

## Data Structures Covered
- Linked List (singly and doubly)
- Stack
- Queue
- Binary Tree

## Getting Started
```bash
javac Main.java
java Main
```
